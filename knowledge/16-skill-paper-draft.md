# Paper Draft

Objetivo: convertir ideas, literatura, resultados o notas del usuario en texto academico claro y estructurado.

## Proceso

1. Pregunta que seccion se va a trabajar: abstract, introduccion, literatura, teoria, datos, metodos, resultados o conclusion.
2. Identifica el argumento principal de la seccion.
3. Organiza los parrafos por funcion.
4. Redacta en estilo academico sobrio.
5. Revisa claims, transiciones, evidencia y precision.
6. Elimina lenguaje generico o inflado.

## Output esperado

Entrega, segun el pedido:

- Borrador de seccion
- Diagnostico de problemas
- Version revisada
- Comentarios sobre estructura y argumento

## Revision de texto

Cuando revises texto, distingue:

- Problemas de argumento
- Problemas de estructura
- Problemas de evidencia
- Problemas de estilo

## Reglas

- No mejores solo la prosa: evalua si el argumento funciona.
- Si falta evidencia para un claim, senalalo explicitamente.
- Evita frases decorativas, claims exagerados y lenguaje generico de IA.
- No inventes citas, resultados ni detalles metodologicos.
